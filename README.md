# Aplikacja_web_TATRY
Celem aplikacji jest przedstawienie i zapoznanie użytkownika z pasmem Tatr oraz szlakami, jakie są dostępne. Użytkownik poprzez interakcje z interfejsem ma możliwość filtrowania pasujących mu szlaków według takich kryteriów jak trudność czy długość oraz sprawdzania szczegółów dotyczących trasy. 

Architektura systemu i narzędzia: 

- Frontend: HTML, CSS3 (Bootstrap + custom częściowo), JavaScript 
- Backend: ASP.NET Core 8.0 MVC, C#; 
- Baza danych: Entity Framework Core 8.0 połączone z Microsoft SQL Server.

Aby uruchomić aplikację w środowisku lokalnym niezbędne jest zainstalowane środowiko Visual Studio 2022 i .NET 8.0 oraz dostęp do lokalnej bazy Microsoft SQL Server. Dokładna instrukcja znajduje się w pliku pdf.
